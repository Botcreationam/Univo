# UNIVO | Empowering Students Beyond the Classroom

> A clean, minimal, premium business showcase website for UNIVO, a student-focused digital platform connecting students with educational opportunities, career guidance, scholarships, and innovation programs across Africa.

## Design Philosophy

Inspired by **Vercel**, **Linear**, **Stripe**, **Notion**, and **Apple**. The design is restrained, professional, and conversion-focused: deep navy, soft blue, subtle cyan accents, and neutral grays. No clutter, no excessive gradients, no flashy animations. Just clean typography, consistent spacing, and meaningful motion.

## Features

- **Hero** with animated headline, value proposition, and clear CTAs
- **About** with mission, vision, values, and animated timeline
- **Services** with 6 core student offerings
- **Impact** with animated count-up statistics
- **Features** with alternating scroll-revealed blocks
- **Student Stories** with auto-playing testimonial carousel
- **Vision** with a 4-phase roadmap
- **Contact** with validated, accessible form
- **Dark / Light mode** with system preference detection
- **SEO** with full metadata, OpenGraph, and Twitter cards
- **Accessibility**: semantic HTML, keyboard navigation, focus styles, reduced-motion support
- **Responsive** mobile-first layouts

## Getting Started

```bash
# Clone
git clone https://github.com/Botcreationam/Univo.git
cd Univo

# Install
npm install

# Develop
npm run dev

# Build for production
npm run build && npm start
```

## Deployment

This project is optimized for Vercel. Push to GitHub and import at [vercel.com](https://vercel.com). No environment variables required.
